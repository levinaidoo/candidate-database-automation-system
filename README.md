# Candidate Database Automation & Tracking System

## Overview
Developed an automated candidate database management and tracking system for a recruitment company using Microsoft Excel, Power Query, Pivot Tables, and advanced lookup automation.

This project was designed to solve issues related to fragmented candidate records, missing database entries, and inefficient manual tracking processes.

---

## Business Problem
The original candidate database contained:
- inconsistent candidate numbering,
- missing candidate records,
- large numbering gaps,
- incomplete candidate details,
- and no structured operational tracking system.

Management could not easily:
- monitor candidate completion progress,
- track work completed by staff,
- identify missing records,
- or generate daily reporting insights.

Additionally, candidate CV folders existed separately from the database, causing inconsistencies between stored CVs and spreadsheet records.

---

## Solution
Designed and implemented a structured and refreshable master candidate database system.

The solution included:
- rebuilding and normalizing candidate numbering from 1–6000,
- automating candidate data population using VLOOKUP, IFERROR, and dynamic COLUMN() formulas,
- extracting candidate information directly from CV folder structures using Power Query,
- automating future database updates through refreshable folder queries,
- implementing status tracking and completion monitoring,
- adding conditional formatting and slicers for operational reporting,
- and creating pivot-based dashboards for daily progress tracking.

---

## Features
- Automated candidate lookup and data population

- Refreshable candidate database linked to CV repositories
- Status tracking workflow

## Outcome
Successfully transformed a fragmented candidate database into a structured, scalable, and refreshable tracking system.

The final solution allows management to:
- monitor candidate completion progress,
- track daily operational activity,
- identify missing candidate records,
- and maintain alignment between CV folders and database records.

The system significantly reduced manual processing effort and improved operational visibility and data governance.
